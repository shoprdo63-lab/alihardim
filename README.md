# מגן סופר - AliExpress חרדי

חנות אונליין צנועה עם 10,000+ מוצרים מאלי אקספרס לציבור החרדי.

## תכונות

- ✅ 10,000+ מוצרים עם קישורי AliExpress
- ✅ סינון צניעות מלא - אין נשים, אין בגדי נשים
- ✅ עיצוב מודרני ורספונסיבי
- ✅ עברית מלאה (RTL)
- ✅ קישורי שותף לכל מוצר

## טכנולוגיות

- Python Flask
- SQLAlchemy
- Bootstrap 5 RTL
- Vercel Serverless
